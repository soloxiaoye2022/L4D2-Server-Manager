#### 教程
1.  先把spcomp.sh上传到服务器
2.  运行spcomp.sh脚本，先输入
```bash
chmod -R 777 steamcmd
```
给予文件夹与子文件夹运行权限（如果已经有给予权限可跳过），再输入
```bash
./spcomp.sh
```
运行脚本

3.  按照提示，输入插件目录和插件名字

#### 使用说明
只输入文件名形式：
1.  必须把`spcomp.sh`放到`*/left4dead2/addons/sourcemod/scripting`这个路径中（`*`指的是你服务端文件的地方）
2.  运行脚本按照提示进行操作即可

输入路径和文件名形式：
1.  脚本可以随意放到任意一个文件夹内
2.  运行脚本按照提示进行操作即可

#### 注意事项
1.  必须把需要编码的sp文件放到`*/left4dead2/addons/sourcemod/scripting`这个路径中（`*`指的是你服务端文件的地方），这是插件平台文件放sp文件的文件夹，也是必须在此文件夹才能进行编码（其他文件夹不行）
2.  脚本需要输入的必须是绝对路径和绝对文件名（不懂就百度）

#### 补充
1.  `bat`后缀格式是`win`系统的脚本，此脚本只有`输入路径和文件名形式`的形式，没有和sh脚本一起更新（因为懒，而且win系统本身就方便）
2.  `sh`后缀格式是`linux`系统的脚本
